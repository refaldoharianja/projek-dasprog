# Program Perhitungan Rangkaian Listrik dan Konversi Bilangan
Program ini merupakan aplikasi berbasis bahasa C yang dibuat untuk melakukan perhitungan rangkaian listrik dasar serta konversi bilangan. Program dirancang menggunakan konsep modular dengan pemisahan file header (`.h`) dan source code (`.c`), serta dilengkapi menu interaktif agar mudah digunakan.

## Tujuan Program
- Menerapkan konsep dasar pemrograman C
- Menggunakan fungsi dan header file secara terstruktur
- Memahami perhitungan listrik dasar
- Memahami algoritma konversi bilangan dengan metode sisa bagi

## Fitur Program
### Perhitungan Listrik Dasar
1. Kalkulator Hukum Ohm  
2. Kalkulator Daya Listrik  
3. Perhitungan Resistor Seri  
4. Perhitungan Resistor Paralel  

### Konversi Bilangan
5. Konversi Bilangan Desimal ke:
   - Biner
   - Oktal
   - Heksadesimal
6. Konversi Bilangan Biner ke Desimal
7. Konversi Bilangan Oktal ke Desimal
8. Konversi Bilangan Heksadesimal ke Desimal

## Cara Menjalankan Program
Pastikan seluruh file berada dalam satu folder.

## Cara Kerja Singkat Program
- Program dijalankan melalui menu utama
- Pengguna memilih menu sesuai kebutuhan
- Program memproses input dan menampilkan hasil perhitungan
- Setiap fitur diletakkan pada file terpisah agar mudah dipahami dan dikembangkan
